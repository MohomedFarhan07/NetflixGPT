## About the Project

* Developed a full-stack movie discovery application using React, Vite, and Firebase, integrating the TMDB API to deliver dynamic and infinite-scroll movie listings.
* Optimized Core Web Vitals by reducing LCP to 0.22s, CLS to 0, and INP to 40ms, significantly improving user experience and performance. 
* Integrated Google Gemini API to provide AI-powered intelligent movie search and recommendations.
* Implemented real-time error tracking, logging, and performance monitoring using Sentry, enabling proactive issue detection and alerting.
* Built comprehensive testing coverage including Unit & Integration tests with Vitest and End-to-End (E2E) testing using Puppeteer.
* Improved accessibility by implementing WCAG-compliant ARIA attributes, ensuring inclusive and screen-reader-friendly user interactions.
* Enabled offline functionality through Progressive Web App (PWA) setup and custom Service Worker caching strategies.
